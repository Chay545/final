# Curavida Home Health (local copy)

Build

```
npm install
npm run build
```

The `build` script compiles Tailwind from `style.css` into `output.css`.

Deploy

- Configure your deployment (Vercel/GitHub) to run `npm run build` before serving.

---

## Reference

This project uses Tailwind CSS. For full documentation and contribution guidelines see [tailwindcss.com](https://tailwindcss.com) and the Tailwind CSS repository.
